GridWorld MDP Solver 🧩

  • Source Available Upon Request
  • Nov 2025
  • Markov Decision Processes

Overview

The GridWorld MDP Solver project focused on solving a grid-based decision-making problem using Markov Decision Processes. The environment represented states, actions, rewards, and transitions, and the goal was to compute the best policy for an agent moving through the grid.

Instead of hard-coding the best path, the agent used value iteration to estimate the long-term value of each state. The solver repeatedly updated state values based on rewards, transition probabilities, and future expected value until the values converged.

This project helped me understand how AI agents can make decisions when outcomes are uncertain and when short-term rewards need to be balanced with long-term results.

What I Built

I implemented the core logic for solving a GridWorld Markov Decision Process, including value iteration, reward evaluation, transition handling, and policy extraction.

I also worked through how different rewards, discount factors, and transition probabilities affect the final policy. This made the project useful for understanding not only how an MDP solver works, but also why an agent chooses one action over another.

Key Features

• Value iteration for estimating the utility of each grid state
• Markov Decision Process modeling with states, actions, rewards, and transitions
• Policy extraction from learned state values
• Discount factor logic for balancing immediate and future rewards
• Convergence-based updates for stable value estimates
• GridWorld environment representation for decision-making tasks
• Analysis of how reward structures affect agent behavior

Impact

This project strengthened my understanding of how reinforcement learning and planning algorithms are built from the ground up. It helped me see how an agent can make rational decisions by reasoning over future outcomes instead of only reacting to immediate rewards.

It also gave me more practice debugging algorithmic AI systems, especially when small changes to rewards or transition probabilities caused large changes in the final policy.
